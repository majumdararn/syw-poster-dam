	% \anode{0}{0}{74cm}{
 %    This poster demonstrates a practical workflow for creating fully reproducible research articles using \emph{showyourwork!}~\cite{Luger2021}. The framework hosts analysis scripts, software requirements, and the manuscript in a GitHub repository, enabling transparent and reproducible research articles. An independent investigator can clone the repository to fully reproduce the scientific article. In the generated manuscript, each figure caption contains two links: one to the dataset stored on Zenodo used to produce the figure, and another to the script that generated it. In addition, \emph{showyourwork!} supports editing through Overleaf, facilitating collaborative scientific writing. (see figure below)
