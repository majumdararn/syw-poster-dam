\begin{textblock}{86}(7.5,35)
\SECTION{Showyourwork philosophy for reproducing scientific articles}
\begin{tikzpicture}
	\anode{0}{0}{22cm}{
    \SUBSECTION{Philosphy}
    
    \emph{Showyourwork!}~\cite{Luger2021} is an open-source reproducibility framework designed to enable transparent and fully reproducible scientific research. It organizes analysis scripts, software dependencies, and the manuscript within a GitHub repository, allowing any independent researcher to clone the project and regenerate the complete article. The manuscript preparation can be integrated with Overleaf, enabling collaborative writing and seamless synchronization between analysis and documentation.

    Once reproduced, figures can be modified directly by editing the underlying scripts that generate them. Raw datasets are typically accessed from a Zenodo repository with read-only permissions, while intermediate and processed data products can be automatically uploaded to a separate Zenodo repository with read–write access.
    }
