\begin{textblock}{86}(7.5,33.5)
\SECTION{Showyourwork philosophy for reproducing scientific articles}
\begin{tikzpicture}
	\anode{0}{0}{22cm}{
    \SUBSECTION{Philosphy}
    
    \emph{Showyourwork!}~\cite{Luger2021} is an open-source reproducibility framework designed to enable transparent and fully reproducible scientific research. It organizes analysis scripts, software dependencies, and the manuscript within a GitHub repository, allowing any independent researcher to clone the project and regenerate the complete article. The manuscript preparation can be integrated with Overleaf, enabling collaborative writing and seamless synchronization between analysis and documentation.

    Once reproduced, figures can be modified directly by editing the underlying scripts that generate them. Raw datasets are typically accessed from a Zenodo repository with read-only permissions, while intermediate and processed data products can be automatically uploaded to a separate Zenodo repository with read–write access.
    }
    \anode{25}{0}{49cm}{
    \begin{figure}
    {
        \script{overview.py}
        \begin{center}
        \includegraphics[width=40cm, trim=0cm 1.3cm 0cm 1cm, clip]{figures/overview.pdf}
        \caption{Framework of \emph{showyourwork!}: GitHub hosts analysis scripts, environment files, dependency rules, and the manuscript; Zenodo stores datasets; and Overleaf enables manuscript editing. Commits trigger automated figure generation and manuscript compilation via GitHub Actions, producing a final version and an arXiv-ready tarball. \label{fig:fig_flw_chart}}    
        \end{center}
    }
    \end{figure}
	}
\end{tikzpicture}
\end{textblock}

\begin{tikzpicture}
    \anode{0}{0}{74cm}{
    \SUBSECTION{Showyourwork framework for this poster}
    
        This poster is created using the \emph{showyourwork!} software, and an electronic copy is available under the Creative Commons Attribution 4.0 International license. The electronic copy can be downloaded from a GitHub repository~\cite{majumdar2026github_syw_poster}, publicly available under the MIT license. Each figure caption in the electronic copy contains a maximum of three icons: The top two icons link to the dynamic and static Zenodo datasets used in the corresponding figure, and the bottom icon links to the script used to make the figure. \\
        
    \SUBSECTION{Application to a use case}

        A dummy use case is proposed in this work to demonstrate the essential features of the showyourwork and its relevance in scientific publications related to photon and neutron experiments. In the proposed use case, raw \emph{measurement amplitude data} is collected using a 1D detector. The raw data, shown in figure \ref{fig:fig_raw_data}, includes a background with linearly dependent on pixel coordinate. This background is removed to produce reduced data, as shown in figure \ref{fig:fig_red_data}. The reduced data is then fitted to $sin(kx+t)$ w.r.t. $k$ and $t$ shown in figure \ref{fig:fig_ana_data}, where $k$ is angular frequency, $t$ is phase shift, and $x$ is pixel positions.
        % This links are shown as icons This work also demonstrates a dummy workflow of raw data processing and analysis through figures \ref{fig:fig_raw_data}, \ref{fig:fig_red_data}, and \ref{fig:fig_ana_data}. The raw data is available at \href{https://doi.org/10.5072/zenodo.471581}{https://doi.org/10.5072/zenodo.471581}~\cite{majumdar2026syw_static_data}, the processed (or reduced) and analyzed data are available at \href{https://doi.org/10.5072/zenodo.470887}{https://doi.org/10.5072/zenodo.470887}~\cite{showyourwork2026syw_dyn_data}
    }
